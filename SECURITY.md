# Паспорт лендинга «Опрос по танцевальным направлениям»

## 1. Общая информация

**Назначение:** Проведение разового внутреннего опроса сотрудников фитнес-клуба о запуске новых танцевальных направлений.

**Стек технологий:** Статический HTML5, CSS3, чистый JavaScript (Vanilla JS). Логические фреймворки и сторонние библиотеки не используются.

**Хостинг / Размещение статики:** GitHub Pages (статический контент, не имеющий доступа к внутренней инфраструктуре компании).

## 2. Архитектура данных и интеграции

**Интеграция:** Обработка формы происходит через платформу автоматизации n8n корпоративного контура.

**Передача данных:** Заполненные пользователем поля отправляются методом POST на webhook n8n и мгновенно перенаправляются на рабочую электронную почту организатора (p.vodyankina@2gis.ru).

**Хранение данных:** Внешние базы данных (включая Google Sheets) не используются. Данные внутри n8n не кэсируются и не сохраняются массивами. Накопление персональных данных на сторонних серверах исключено.

## 3. Информационная безопасность и Веб-аналитика (Критично для ОИБ)

**Веб-аналитика:** Сторонние системы веб-аналитики (Google Analytics, Яндекс Метрика, Amplitude и др.) на сайте полностью отсутствуют. Автоматический сбор логов, поведенческих факторов и кликстрима не ведётся.

**Сбор скрытых идентификаторов:** Система не собирает и не передаёт автоматически какие-либо скрытые параметры пользователей: IP-адреса, MAC-адреса, корпоративные ID (User ID), куки (Cookies) или данные авторизации.

**Персональные данные (ПДн):** Сайт собирает только те данные, которые пользователь вводит в поля формы добровольно (Имя, Email). Передача осуществляется по шифрованному протоколу HTTPS.
